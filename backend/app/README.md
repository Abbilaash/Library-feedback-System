# README.md - Follow the instructions to run the backend

## Installation

```
pip install -r requirements.txt
```

## Setting up ML model

```
cd backend/app
```

```
python setup.py
```

## Setting .env file

```
FLASK_SECRET_KEY = "<some_text>"
MONGO_URI = ""
GMAIL_ID = "<sender_mail_id>"
GMAIL_PASSWORD = "<gmail_application_password>"
```
